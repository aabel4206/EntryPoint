# EntryPoint

EntryPoint is a full-stack web application designed to help new Texas State University students navigate onboarding tasks, academic resources, and campus services through personalized checklists and guided workflows.

## Project Scope

This version focuses on new Texas State University students, with special attention to international students.

## Core Features

- Personalized onboarding checklist
- Categorized Texas State student resources
- Student and administrator roles
- Admin-managed resources and checklist templates
- AI-assisted monitoring of selected Texas State webpages for important changes

## Technology Stack

- React
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT Authentication

## Independent Study Focus

This project combines software engineering with a limited applied AI research component. The AI component investigates how monitoring university webpages can help keep student-facing information accurate and up to date.

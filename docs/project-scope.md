# Project Scope

## Project Name

EntryPoint

## Project Overview

EntryPoint is a full-stack web application designed to help new Texas State University students navigate onboarding tasks, academic resources, and campus services through a centralized and structured platform.

The system focuses on helping students identify important university requirements, locate relevant resources, and complete onboarding activities using personalized checklists and guided workflows.

The project is being developed as part of a CS 4395 Independent Study and includes both a software engineering component and a limited applied AI research component.

## Target Users

The primary users are new Texas State University students.

The system places additional emphasis on international students because they often face additional onboarding requirements involving immigration, advising, housing, registration, and campus resources.

## User Roles

### Student

Students can:

- Create accounts
- Log in
- Access personalized onboarding checklists
- View university resources
- Track task completion

### Administrator

Administrators can:

- Manage resources
- Manage categories
- Manage onboarding tasks
- Review webpage monitoring results
- Maintain system content

## Core Feature

The primary feature of EntryPoint is a personalized onboarding checklist that helps students complete important university tasks in an organized and structured manner.

## Resource Categories

Initial categories include:

- Admissions
- Orientation
- Housing
- Financial Aid
- Academic Advising
- Registration
- International Student Services
- Technology Resources
- Career Services
- Student Organizations

## AI-Assisted Component

The AI-assisted component focuses on monitoring selected Texas State webpages and detecting meaningful content changes.

The purpose is to help ensure that resource information remains current and to reduce the likelihood of students relying on outdated information.

## In Scope

- Student authentication
- Personalized onboarding checklists
- Resource management
- Resource categorization
- Admin dashboard
- Webpage monitoring
- Change detection
- Change summaries
- Documentation and reporting

## Out of Scope

- Support for universities outside Texas State University
- Single Sign-On integration with Texas State systems
- Mobile applications
- Full conversational chatbot functionality
- Large-scale university web crawling
- Production deployment for university-wide usage
